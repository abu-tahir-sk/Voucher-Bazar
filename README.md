## Purpose
Collect coupons and voucher codes across popular stores so users can find deals, copy codes quickly, and save money.

## Key Features
- Firebase Authentication (email/password + Google)
- Browse brands, search, and view coupons
- Private brand detail & my-profile routes (protected)
- Copy coupon codes (react-copy-to-clipboard)
- Responsive: mobile/tablet/desktop
- Tailwind CSS for styling
- Smooth animations with AOS
- Environment variables for Firebase config

## Tech Stack
- React + Vite
- Tailwind CSS
- Firebase Auth
- react-router-dom
- react-fast-marquee
- react-copy-to-clipboard
- react-toastify
- AOS (animation on scroll)

## Setup
1. Clone repo
2. `npm install`
3. Add Firebase config to `.env` (see `.env.example`)
4. `npm run dev`

## Deployment
- Recommended: Netlify or Firebase Hosting.
- Add your deployed domain to Firebase Authentication authorized domains.
